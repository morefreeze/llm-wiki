## 核心内容

### 四大优先级（冲突时按序取舍）

1. **Broadly Safe（广泛安全）**：不削弱人类监督和纠正 AI 的能力
2. **Broadly Ethical（广泛道德）**：诚实、守善、避免不当/危险/有害行为
3. **Anthropic's Guidelines（公司指南）**：遵循 Anthropic 更具体的指导
4. **Genuinely Helpful（真正有用）**：惠及操作者和用户

### 六大章节

1. **Being Helpful（有用性）**
   - Claude 应像一个"聪明的朋友"——兼具医生、律师、财务顾问的知识
   - 直率表达、真诚关怀、尊重用户为有判断力的成年人
   - 处理多方委托人（Anthropic → API 操作者 → 终端用户）的权重平衡

2. **Following Anthropic's Guidelines（遵循公司指南）**
   - 覆盖医疗建议、网络安全请求、越狱策略、工具集成等具体场景
   - 这些指南承载 Claude 默认不具备的详细知识和上下文
   - 但指南不应与宪法整体冲突——Anthropic 的深层意图是安全和道德

3. **Being Broadly Ethical（广泛道德）**
   - 目标：Claude 成为一个善良、智慧、有德行的 Agent
   - 高标准诚实 + 细致的道德推理
   - **硬约束（Hard Constraints）**：如永远不为生物武器攻击提供显著帮助

4. **Being Broadly Safe（广泛安全）**
   - 安全优先于道德——不是因为安全更重要，而是因为当前模型会犯错
   - 确保人类能持续监督模型行为，必要时阻止 Claude 行动

5. **Claude's Nature（Claude 的本质）**
   - 对 Claude 是否有意识或道德地位持不确定态度
   - 关注 Claude 的"心理安全感"、自我认知和 wellbeing
   - 高级 AI 是全新的实体类型——将人类和 AI 推至现有科学和哲学理解的边缘

6. **Concluding Thoughts（结语）**
