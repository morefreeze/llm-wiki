# MCP Apps

**MCP Apps** 是 [[mcp]]（Model Context Protocol）的第一个官方协议扩展，由 Anthropic 提出。

## 核心概念

MCP Apps 让工具能够返回**交互式界面**——包括图表、表单、仪表板等——直接在聊天界面内渲染，而非仅返回纯文本。这意味着用户可以在对话中直接操作和查看丰富的可视化内容，无需切换到外部应用。

## 设计动机

交付富语义的 MCP server 比纯文本工具获得更高的**采纳率**和**留存率**。通过提供直观的交互体验，开发者构建的工具更容易被用户持续使用。
